# eede-term
---
> 온도 센서를 이용한 프로젝트를 진행하였다.

## 구현기능
---
* 온도센서에서 입력받은 값을 섭씨 온도로 환산하여 7-seg에 표시한다.
* 버튼을 조작해 현재 온도값을 캡쳐한다.
